# Test Web Server

This is a minimum test server that can be used for various usage, such as deploying it on cloud to test.

## How to Run

By default, the flags set are:
- ID: `id`
- Port: `80`

```
go run .
```

After the server is running, you can test it using CURL

```
curl <host:ip>
```

```
curl localhost:80
```
